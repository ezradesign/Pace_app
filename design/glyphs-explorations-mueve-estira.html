Especificaciones técnicas

Tokens de color por módulo

Mueve: #9A7B4F
Estira: #6B7A8F

Estados visuales del glifo

Default:  stroke: var(--ink-3) @ 60% opacity;   fill: none;    border: 1.5px dashed var(--line);
Active:   stroke: var(--move);                    fill: var(--move-soft); border: 1.5px solid var(--move);
Completed: stroke: var(--ink-2);                  fill: none;    + check SVG overlay;
Future:   stroke: var(--line) @ 40% opacity;       fill: none;    opacity: 0.35;

API de componente JSX (propuesta)

// MoveGlyph.jsx
function MoveGlyph({ tag, state = 'default', stepIdx }) {
  // tag: 'PUSH' | 'PULL' | 'LEG' | 'CORE' | 'GRIP' | 'POST'
  //      | 'SIT' | 'HIP' | 'SHLD' | 'ATG' | 'ANC' | 'SIT'
  // state: 'default' | 'active' | 'completed' | 'future'
  // stepIdx: fallback a símbolo unicode rotativo si no hay SVG
}

Mapeo tag → glifo

PUSH  → Flexión inclinada      (torso en push-up, líneas de tensión)
PULL  → Fondos en silla         (cuerpo en tracción, flechas de fuerza)
LEG   → Wall sit                (90° pierna, isométrico)
STEALTH→ Gemelos subrepticios   (pantorrilla en esquina)  
CORE  → Hollow hold             (V invertida, músculo central)
GRIP  → Apretón de puño         (antebrazo + puño cerrado)
POST  → Postura reset            (columna + flechas de corrección)
SIT   → Antídoto silla         (figura sentada, flechas de apertura)
HIP   → Caderas                 (articulación, flechas de rotación)
SHLD  → Hombros                 (clavícula + braquial + arcs)
ATG   → ATG lunge               (rodilla por delante del pie)
ANC   → Ancestral               (figura en squat + raíces)
