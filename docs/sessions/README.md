# Diario de sesiones · PACE

Archivo histórico de todas las sesiones de trabajo. Cada archivo contiene el
relato completo de una sesión: qué se hizo, por qué, decisiones tomadas,
archivos modificados.

**Regla:** este directorio es sólo-lectura para sesiones nuevas. No se edita
contenido pasado — si una decisión antigua se invalida, se anota el cambio
en `STATE.md` (sección "Decisiones activas") y se referencia aquí desde la
sesión en que cambió.

Para estado actual del proyecto y próximos pasos, ver [`../../STATE.md`](../../STATE.md).
Para historial público de versiones, ver [`../../CHANGELOG.md`](../../CHANGELOG.md).
